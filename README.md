# BBANG MAP #
> 프랜차이즈 빵집들로 가득한 통합 지도앱과 다르다. \
> 유명 빵집부터 동네 빵집까지, 검증된 빵집들만 보여주는 <b>빵집 전용 지도 애플리케이션</b>

### 주요기능 ###
* 지도 공유 커뮤니티
* 주변 빵집 지도
* 마이 빵집 지도
    * 즐겨찾는 빵집
    * 내가 만든 빵맵
    * 스크랩한 빵맵
### 기술 ###
<b>APIs : </b> 
네이버지도 API, javax.mail
